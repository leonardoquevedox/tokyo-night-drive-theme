## <img src="https://github.com/user-attachments/assets/33bfbe3f-9872-46bd-8454-433a94e19222" height=18 />&nbsp; Installation

1. Open the `Command Palette` with `Ctrl + Shift + P` / `Command + Shift + P`
2. Type `Tokyo Night Drive`
3. Press Enter
4. Enjoy it!
